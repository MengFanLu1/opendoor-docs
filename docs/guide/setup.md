# 快速开始

欢迎使用 OpenDoor AI 中转服务。完成账号注册、充值和 API Key 创建后，就可以在 Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw、Hermes、Cline 等客户端中接入使用。

![快速开始三步](/guide/quick-start-steps.png)

## 第一步：注册账号

访问 [code-opendoor.com](https://code-opendoor.com)，点击右上角「注册」进入注册页面。

注册时填写邮箱、密码和邮箱验证码；如果有邀请码，也可以在注册页面一并填写。

![注册账号](/guide/quick-start-register.png)

## 第二步：充值

登录后进入控制台，点击顶部导航里的「充值」。

选择充值金额和支付方式后完成支付。充值到账后，余额会同步到控制台，用于后续模型调用消费。

![充值](/guide/quick-start-recharge.png)

## 第三步：获取 API Key

进入「API Key」页面，点击「创建密钥」。

创建时建议填写一个容易识别的名称，例如 `claude-code`、`codex`、`cursor`。如果你想控制单个 Key 的消费上限，可以取消「无限制」并填写额度限制。

![创建 API Key](/guide/quick-start-apikey-create.png)

创建成功后，完整 API Key 只会显示一次。请立即复制并妥善保存，关闭弹窗后无法再次查看完整密钥。
